# Sistema de Gestión de Pólizas y Seguros

## Descripción

Aplicación web empresarial desarrollada con ASP.NET Core MVC para la administración de clientes, pólizas, tipos de seguros, usuarios y siniestros.

Este proyecto tiene como objetivo fortalecer conocimientos en tecnologías .NET utilizadas en entornos corporativos, incluyendo Entity Framework Core, SQL Server, arquitectura MVC e inyección de dependencias.

---

## Funcionalidades

### Gestión de Clientes

* Crear clientes
* Consultar clientes
* Editar clientes
* Eliminar clientes

### Gestión de Pólizas

* Administración de pólizas
* Control de vigencia
* Asociación con clientes

### Gestión de Siniestros

* Registro de siniestros
* Seguimiento de estados

### Gestión de Usuarios

* Administración de roles
* Control de acceso

## Arquitectura

El proyecto sigue el patrón MVC:

* Models
* Views
* Controllers

Además utiliza Entity Framework Core como ORM para la persistencia de datos.

## Base de Datos

Entidades principales:
